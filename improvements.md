# Future changes to the project

## New features

- New tool: convert image into icon (.ico)
- New tool: rotate, mirror, flip image
- New tool: strip metadata
- New tool: remove background (output in png)
- The output image of one tool can be used as input for another tool (tool chain)


## Modifications

- Output format is always the same as the input format (except for the remove background tool).
- Output quality is always 100% (except for the compress tool obviously)
